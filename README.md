# Machine Learning Projects

A portfolio of machine learning projects covering classification, regression, and clustering — built with Python and scikit-learn.

---

## Projects

| # | Project | Category | Algorithm(s) | Description |
|---|---------|----------|--------------|-------------|
| 1 | [Dry Bean Classification](./Dry%20Bean%20Classification) | Multiclass Classification | Logistic Regression | Classifies dry beans into 7 varieties using geometric and shape-based features |
| 2 | [Patient Health Risk Prediction](./Pateint%20Health%20Risk%20Prediction) | Regression | Linear, Lasso, Ridge | Predicts patient health risk scores based on vitals, lifestyle, and biological markers |
| 3 | [Social Network Ads](./Social%20Network%20Ads) | Binary Classification | Logistic Regression | Predicts purchase behavior of social network users based on demographics |
| 4 | [Students Score](./Students%20Score) | Polynomial Regression | Polynomial Regression (degree 3) | Predicts student exam scores from study hours |
| 5 | [Weather Classification](./Weather%20Classification) | Classification | — | Classifies weather conditions based on meteorological features |
| 6 | [Weather Temperature Prediction](./Weather%20Temprature%20Prediction) | Linear Regression | Linear Regression | Predicts daily temperature using sunlight hours and humidity |
| 7 | [Titanic Survival Prediction](./Titanic%20Survival%20Prediction) | Binary Classification | Gaussian Naive Bayes | Predicts passenger survival using features like class, sex, age, and fare |
