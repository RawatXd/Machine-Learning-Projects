# Machine Learning Projects

A collection of hands-on ML projects arranged from basics to advanced concepts.

## Projects

| # | Project | Type | Description |
|---|---------|------|-------------|
| 1 | [Students Score](./Students%20Score) | Simple Linear Regression | Predicts exam scores from study hours — the most fundamental ML starting point |
| 2 | [Weather Temperature Prediction](./Weather%20Temprature%20Prediction) | Multiple Linear Regression | Predicts daily temperature from sunlight hours and humidity using multiple features |
| 3 | [Patient Health Risk Prediction](./Pateint%20Health%20Risk%20Prediction) | Regularized Regression | Predicts health risk scores using Linear, Lasso & Ridge Regression with regularization |
| 4 | [Social Network Ads](./Social%20Network%20Ads) | Binary Classification | Predicts purchase behavior from user demographics using Logistic Regression |
| 5 | [Titanic Survival Prediction](./Titanic%20Survival%20Prediction) | Binary Classification | Predicts passenger survival using classification models on the Titanic dataset |
| 6 | [Dry Bean Classification](./Dry%20Bean%20Classification) | Multiclass Classification | Classifies 7 dry bean varieties using Logistic Regression on geometric features |
| 7 | [Weather Classification](./Weather%20Classification) | Multiclass Classification | Classifies weather types based on meteorological features |
| 8 | [Decision Pet Adoption](./Decision%20Pet%20Adoption) | Decision Tree | Predicts pet adoption outcomes using a Decision Tree classifier |


## Tech Stack

Python · NumPy · Pandas · Scikit-learn · Matplotlib · Seaborn · Jupyter Notebook
