## 👋 About
 
Welcome to my Machine Learning Projects repository! This portfolio contains a collection of hands-on projects demonstrating various ML techniques — from data cleaning and preprocessing to model training, evaluation, and interpretation.
 
Each project is self-contained with its own dataset, notebook, and documentation.

## 🚀 Getting Started
 
### 1. Clone the repository
```bash
git clone https://github.com/RawatXd/Machine-Learning-Projects.git
cd Machine-Learning-Projects
```
 
### 2. Install dependencies
```bash
pip install pandas numpy scikit-learn matplotlib seaborn notebook
```
 
### 3. Open any project
```bash
cd "Titanic Survival Prediction"
jupyter notebook
```
 
> Each project folder contains its own dataset and notebook — just open and run!

## 📈 Skills Demonstrated
 
- **Data Cleaning** — handling missing values, outliers, and inconsistencies
- **EDA** — exploratory analysis with visualizations (histograms, bar charts, pie charts, heatmaps)
- **Feature Engineering** — encoding, scaling, and selecting relevant features
- **Model Training** — supervised learning with classification and regression algorithms
- **Model Evaluation** — accuracy, precision, recall, F1-score, confusion matrix, R²
- **Regularization** — Lasso and Ridge regression to prevent overfitting
